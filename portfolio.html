Python Program for Automated Content Migration

I wrote a Python program to migrate content between content management systems used by Baystone Media. The program helped upgrade hundreds of websites. Using the popular Requests and Beautiful Soup libraries, the program parses HTML content from legacy client websites; it then uses this content to create a website on the newer platform automatically. (Project Year: 2017)

Arduino Antenna Rotator Controller PCB

I designed a printed circuit board for use in the Adler Planetarium's radio communication ground station. Controlled via computer software, it tilts the planetarium's antenna in order to acheive the best signal. I helped set up the computer software, and I helped write the Arduino sketch. This PCB design is a modification of the K3NG Rotator Controller. I saved the planetarium money. This PCB costs only about $45 (PCB, Arduino, and other necessary electrical components); this is far cheaper than similar technology that cost hundreds sold by vendors. (Project Year: 2016)
